# Controle de feira
Aplicativo WPF para auxiliar no controle de feira e gerenciamento dos produtos da dispensa de casa. Guarda registro das compras; ex: nome do produto, marca, data da compra, local da compra e  preço; E também guarda informações dos produtos que têm na dispensa, em casa, para que no futuro o usuário tenha controle sobre suas compras e possa analisar a necessidade e a rentabilidade para efetuar novas compras.
